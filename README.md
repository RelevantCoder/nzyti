# install


install zorich to be default shell :
<br>
$ cd install
<br>
$ chmod +x install-Def.sh
<br>
$ ./install-Def.sh
<br>
<br>
or install global :
<br>
$ cd install
<br>
$ chmod +x installer.sh
<br>
$./installer.sh
<br>
<br>
normal use  :
$ python3  main.py
